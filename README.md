# Flight Weather Analyzer

This project is a simple Flask-based weather application that allows users to search for a city and view its current weather conditions.

## Features
- Search weather by city
- Automatic default city on page load
- Error messages for invalid input
- Flight suitability evaluation based on wind speed
- Clean and simple interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- OpenWeather API

## Installation
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Create a `.env` file and add your OpenWeather API key:
   API_KEY=your_api_key
4. Run the application:
   py app.py

## Usage
Open your browser and go to:
http://127.0.0.1:5000
